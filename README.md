# Smart Upgrade IT Website

Local editable version of `https://www.smartupgradeit.com/`.

## Folder Layout

- `index.html` - Vite entry file
- `src/main.jsx` - React mount point
- `src/App.jsx` - top-level React app
- `src/pages/` - page components such as Home, Services, About, Contact
- `src/components/` - reusable React sections
- `src/assets/styles/` - CSS files
- `src/assets/images/` - logos, photos, icons, and other media
- `docs/` - planning notes, copy drafts, and site references

## How To Preview

Install dependencies once:

```powershell
npm.cmd install
```

Run the React dev server:

```powershell
npm.cmd run dev
```

Then visit the local URL printed by Vite.

## Build

```powershell
npm.cmd run build
```
